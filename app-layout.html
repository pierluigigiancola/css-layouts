App Layout Header and Body

A cointainer has a fixed height (in this case the container is the body and the height is100vh) and it uses display: grid to split the available space in 2 rows.

The first row is a header with a fixed height (in this case min-content) and the second row is the main content which use all the remaining space in the container height.

The second row sets overflow: auto to scroll when the content overflow
